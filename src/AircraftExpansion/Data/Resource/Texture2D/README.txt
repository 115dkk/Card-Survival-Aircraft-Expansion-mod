Place card images here.

File naming convention:
- Imp_AircraftExpansion.png - Aircraft Expansion upgrade card image

Image requirements:
- Format: PNG, JPG, or JPEG
- The filename (without extension) becomes the sprite name
- This sprite name is referenced in CardData JSON via CardImageWarpData

Example:
If you place "Imp_AircraftExpansion.png" here, the JSON should have:
  "CardImageWarpData": "Imp_AircraftExpansion"
